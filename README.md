This is a simple calculator web application built using HTML, CSS, and JavaScript. It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features

- **AC (All Clear):** Clears the display.
- **DE (Delete):** Deletes the last character from the display.
- **Decimal Point ("."):** Appends a decimal point to the display.
- **Division ("/"):** Appends a division symbol to the display.
- **Numeric Buttons (0-9):** Append their respective values to the display.
- **Equal Button ("="):** Triggers the calculation and displays the result.

## How to Use

1. Clone this repository to your local machine.
2. Open the `index.html` file in your web browser.
3. Use the buttons to input numbers and perform calculations.
4. Click the "=" button to see the result.
